# Bridge Configuration Reference（已迁出）

`ilink-hub-bridge` 的配置参考已随 bridge 一并迁入独立项目
[`jeffkit/im-agentproc`](https://github.com/jeffkit/im-agentproc)。

本仓库（`ilink-hub`）自 `0.4.0` 起不再包含 bridge 代码与 `ilink-hub-bridge` 二进制，
bridge 的 YAML profile 规范、`agentproc:` 字段、`{{MESSAGE}}` 占位符等配置说明请到
im-agentproc 仓库查阅最新版本。

历史背景见 [`docs/proposals/bridge-as-multi-im-runtime.md`](https://github.com/jeffkit/ilink-hub/blob/main/docs/proposals/bridge-as-multi-im-runtime.md) 附录 A。
