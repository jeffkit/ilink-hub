WeChat Commands

/list · /ls 列出所有已注册后端及其状态
/use <name> · /switch 切换当前活跃后端（如 /use mac-home）
/broadcast <text> · /all 向所有在线后端广播消息
/status 查看 Hub 状态（在线/总客户端数）
/help · /? 显示帮助信息
/session list · /session ls 列出当前对话的所有 sessions
/session new [name] [uuid] 新建 session（名称可选，省略则自动生成；可附带初始 UUID）
/session use <name> 切换到指定 session
/session delete <name> · /session rm · /session del 删除指定 session（不能删除当前活跃的）
